# JobTracker

This repository contains the Smart Job Tracker application.

The app source code and full product documentation are inside the [jobtracker](jobtracker) folder:

- App docs: [jobtracker/README.md](jobtracker/README.md)
- App source: [jobtracker/src](jobtracker/src)

## Quick Start

```bash
cd jobtracker
npm install
npm run dev
```

## Build

```bash
cd jobtracker
npm run build
npm run preview
```
